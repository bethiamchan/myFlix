myFlix Documentation


About

This project is a REST API that interacts with a MongoDB database. It gives users access to information about movies, directors, and genres. Users can sign up, update their information, and create a list of their favorite movies.


Accessing Movie Collection

HTTP Method URL Request Request Body Format Response Body Format
GET /movies Get a list of all movies None JSON object with data about all movies
GET /movies/:Title Get data about a single movie by title None JSON object with data about a single movie
GET /movies/genres/:Genre Get data about a genre by genre name None JSON object with data about a single genre
GET /movies/directors/:Name Get data about a director by name None JSON object with data about a single director

Accessing User Collection

HTTP Method URL Request Request Body Format Response Body Format
POST /users Register as new user JSON object with new user details JSON object with data about added user
PUT /users/:Username User updates their user information JSON object with updated user details JSON object with data about updated user
POST /users/:Username/movies/:MovieID User adds movie to list of favorites None JSON object with user data including movie ID that was added to favorites
DELETE /users/:Username/movies/:MovieID User removes movie from list of favorites None JSON object with user data showing movie ID that was removed from favorites has been removed
DELETE /users/:Username Existing user deregisters None Message indicating account has been deleted or was not found
