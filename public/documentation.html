myFlix Documentation

This app will give user access to information about movies, directors, and genres. Users can sign up, update their information, and create a list of their favorite movies.

“Genre”: “Comedy”,

}

Register as new user /users POST JSON {
“Username”: “myusername”,
“Password”: “myPassword123”,
“Email”: myname@sample.com
} 		JSON object with data about user {
“Username”: “myusername”,
“Password”: “myPassword123”,
“Email”: myname@sample.com
}
User updates their user information /users/:username PUT JSON {
“Username”: “mynewusername”,
“Password”: “myNewPassword123”,
“Email”: myothername@sample.com
} 		JSON object with data about user {
“Username”: “mynewusername”,
“Password”: “myNewPassword123”,
“Email”: myothername@sample.com
}
User adds movie to list of favorites /users/:username/movies/:movieID POST JSON {
"Favorite Movies": "87cf4ea5"
} 		JSON object with data about movie that was added to favorites {
"Favorite Movies": "87cf4ea5"
}
User removes movie from list of favorites /users/:username/movies/:movieID DELETE None /users/myusername/movies/87cf4ea5 Message indicating movie was removed successfully “Ladies in Black was removed from your favorites.”
Existing user deregisters /users/:username DELETE None /users/myusername Message indicating account has been deleted “Your account has been deleted from myFlix.”

Note: Use %20 for spaces in the URL
